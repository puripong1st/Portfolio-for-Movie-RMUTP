" เด็กติดเกม ที่ได้เขียนโค้ด "

เด็กติดเกม จากเด็กติดเกมที่ได้หันหน้าเข้าสู่วงการเกมออนไลน์และได้มีโอกาสเขียนโค้ดเอง แค่ใครในอดีตได้ยินคำนี้แล้วก็คิดว่าไม่มีอนาคตแน่ๆ จนข้าพเจ้าได้หันหน้าเข้าสู่วงการ การเขียนโค้ดครั้งแรกตอน ป.6 ข้าพเจ้าได้เริ่มจากการเขียนโค้ดภาษา JAVA ในเกมมายคราฟ ซึ่งข้าพเจ้าได้มารับรู้ที่หลังว่าภาษา JAVA เป็นภาษาที่ยากที่สุด ย้อนไปเมื่อ 6 ปีแล้ว ยูทูป แทบจะไม่มีสอนเขียนภาษา java ในมายคราฟเลยสักคนมากสุดก็แค่เป็นโปรแกรมที่ให้เรา วางรูป png ไปเพื่อให้แสดงเป็นดาบบล็อค 2d เท่านั้น ในตอนนั้นที่ข้าพเจ้าเริ่มเขียน java ในมายคราฟเป็นเวอร์ชั่น 1.7.10 ตอนนั้นคนโตๆที่เป็นรุ่นพี่เทียบเท่ามหาลัย หรืออาจจะมากกว่านั้น ได้มีการเขียน mod ที่เป็น 3d ขึ้นเทียบเท่าเกม mmo ของเกมยุคนั้นเลยก็ว่าได้ ข้าพเจ้ารู้สึกสนใจอย่างยิ่ง จนได้เริ่มเขียนโค้ดมาเรื่อยๆ ทำได้บ้างไม่ได้บ้าง ข้าพเจ้ารู้สึกเริ่มช้าไปในการเขียน java ในตอนนั้น เพราะกระแสเซิฟเวอร์มายคราฟที่เป็น mod เริ่มดับสนิท และเป็นการมาของ roblox และยังมีโทรศัพท์อีก ทำให้ข้าพเจ้าได้หยุดติบทบาทในการเขียนโค้ด java จนกระทั่งข้าพเจ้าได้ค้นพบ fivem อีกรอบในภาษา lua ข้าพเจ้าได้ทำการเปิดเซิฟแรกที่มีชื่อว่า dragonpixel เป็นชื่อเดียวที่เคยเปิดเซิฟมายคราฟ เซิฟนี้ถูกเปิดเมื่อ 3 ปีที่แล้ว สามารถเข้าชมในยูทูปได้เพียงค้นหาคำว่า "GTA V : Roleplay - โปรโมทเซิฟ FiveM DragonPixel เปิดใหม่แจกเงินตั้งตัว 5M รถวิ่ง 540 โหดเกิ๊นนน" ข้าพเจ้าได้ศึกษาภาษา lua มาอย่างต่อเนื่องแทบจะไม่หยุดพักจนกระแส fivem ในวงการ dev เริ่มมีมากขึ้นเรื่อยๆข้าพเจ้าถึงได้วางมือ และหันหน้าเข้าสู่วงการ Web Developer จนถึงปัจจุบัน และข้าพเจ้านั้นอยากทำการศึกษาต่อที่ คณะครุศาสตร์อุตสาหกรรม สาขาวิชาคอมพิวเตอร์และอิเล็กทรอนิกส์อัจฉริยะ มหาวิทยาลัยเทคโนโลยีราชมงคลพระนครเพราะตัวข้าพเจ้าคิดว่า มันเป็นโอกาสครั้งหนึ่งที่จะได้พัฒนาตัวเองในการศึกษาเพราะการศึกษาในรั้วของโรงเรียนนั้น ไม่ได้สอนข้าพเจ้าในการเขียนโค้ดเลยสักครั้ง ข้าพเจ้าได้ทำการเรียนรู้ด้วยตัวเองมาด้วยตลอด และได้มีการซื้อคอร์สเรียนออนไลน์เพื่อเป็นการขัดเกลาความรู้ตัวเอง ว่ามีความสามารถหรือว่าไม่มี และตัวข้าพเจ้านั้นก็ไม่หยุดที่จะเรียนรู้ เพื่ออนาคตของตัวเอง ครอบครัว ประเทศ เพื่อว่าโปรแกรมเล็กๆ ที่ข้าพเจ้าได้เขียนขึ้นเป็นโค้ด อาจจะมีน้องๆ ได้นำชุดโค้ดของข้าพเจ้าที่ได้ทำการเขียน และ ปรับปรุง ไปศึกษาต่อ และอาจจะต่อยอดได้สูงสุด หากตัวข้าพเจ้านั้นได้เข้าเรียนต่อที่ คณะครุศาสตร์อุตสาหกรรม สาขาวิชาคอมพิวเตอร์และอิเล็กทรอนิกส์อัจฉริยะ มหาวิทยาลัยเทคโนโลยีราชมงคลพระนครข้าพเจ้านั้นได้ฝันว่า ได้ทำโปรเจก Start up และ มีชื่อเสียงในด้านโปรแกรม และก่อตั้งบริษัท หรือ ได้เข้าบริษัทดีๆ ตัวข้าพเจ้าทราบดี ว่าเกรดเฉลี่ยของตัวข้าพเจ้าเองอาจจะไม่ถึงเกณฑ์ แต่อย่างน้อยๆ portfolio ของตัวข้าพเจ้าก็ได้โชว์ว่า ตัวข้าพเจ้านั้น มีความรู้อะไรบ้าง และทำให้ข้าพเจ้าได้รวมรวบ ความสามารถทั้งหมดมาลงใน portfolio นี้ ขอบคุณ คณะกรรมการ หรือ คณะอาจารย์ ที่ได้ดู portfolio ของตัวข้าพเจ้า
